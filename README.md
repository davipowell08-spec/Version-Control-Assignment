Version control is a system used to track and manage all versions of a project. 
Git is the intermediary action of dragging the file to github which retains this file permanently on the internet
Git status shows the status of your file, so if any changes has been made or what is in process.
Git add adds the file on computer
Git commit -m stores the file on computer
Git push pushes the file from the computer to the cloud (attached respository to folder)
